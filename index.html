Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language, and it is the standard markup language for creating web pages.

HTML describes the structure of a web page using elements that are represented by tags.

HTML Elements

Common HTML Elements

HTML elements are the building blocks of HTML pages. They include:

  • Headings: <h1> to <h6>
  • Paragraphs: <p>
  • Links: <a>
  • Images: <img>
HTML Attributes

What are HTML Attributes?

Attributes provide additional information about HTML elements. They are always specified in the opening tag, and come in name/value pairs like name="value".

HTML Forms

Creating Forms in HTML

HTML forms are used to collect user input. They are defined using the <form> element and can contain various input elements like text fields, checkboxes, and buttons.
